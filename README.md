# Turkish Hate Speech Detection (News Domain)

This repository contains the experimental code for the NLP course project.

The study focuses on hate speech detection in Turkish news texts using a
Transformer-based model.

## Model
- dbmdz/bert-base-turkish-cased

## Dataset
- TurkishHatePrintCorpus

## Evaluation
- Accuracy
- Macro F1-score

## Environment
- Python 3.12
- PyTorch (MPS backend on Apple Silicon)

The notebook reproduces the results reported in the project report.
